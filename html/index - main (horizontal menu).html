14 Popup Windows











Success

Pop-up windows that are used to provide positive confirmations or information should use the same colors and styles for the header as Success Alerts (or notifications).

For example, the user submits a change to a document or file and the system detects no issues (e.g. formatting, connectivity, etc.), then the user might be presented with a Success message.

Info

Pop-up windows that are used to provide additional information that is not necessarily positive or negative should use the same colors and styles for the header as Info Alerts (or Notifications).

For Example, if the user needs to see information about an upcoming action but not (at least not yet) see the positive or negative impact of that action, then an Info pop-up might be used.

Warning

Pop-up windows that are used to provide additional information that an action or a state is negative or could become negative but isn't necessarily destructive.

For example, deleting something that is easy to retrieve, is easy to recreate, or is fairly non-essential might constitute a Warning. Deleting important data from a database that is mission critical or that could not be retrieved might warrant an Error.

Error

Pop-up windows that are used to provide additional information that an action or a state is negative or could become negative in a way that is destructive or potentially destructive, particularly if such an action is difficult to undo.

For example, deleting something that is easy to retrieve, is easy to recreate, or is fairly non-essential might constitute a Warning. Deleting important data from a database that is mission critical or that could not be retrieved might warrant an Error.

Neutral

Neutral Pop-up windows could be used to present information or functionality that does not meet the recommendations or requirements for other pop-up types.
